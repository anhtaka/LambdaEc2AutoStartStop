# set up Lambda 

```cmd
git clone APP_URL
npm install / update
zip -rq UPFIE_NAME.zip node_modules/ app.js
```
up file to Lambda

for Docker
```cmd
docker-compose up
docker-compose run app npm install
docker-compose run app zip -rq UPFILE_NAME.zip node_modules/ app.js
```
